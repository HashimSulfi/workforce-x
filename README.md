# WorkForce X – Catering Team Marketplace System

A web-based platform to connect customers with catering teams.

## Features
- Role-based system (Admin, Customer, Team)
- Booking workflow: Requested → Accepted → Completed → Rejected
- Rating and review system
- Search teams by district

## Technologies
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Description
This project solves the problem of unorganized catering booking by providing a centralized system.
